# tupo

`tupo` is a command line script that adds a typo into your JavaScript.

* Install: `npm install tupo -g`
* Usage:

        Usage: tupo [options] <JS file>

        Options:

          -h, --help         output usage information
          -t, --type [type]  Type of typo

        Available types:

          syntax
          quotes
          comma
          binaryOperator
          mismatch
          keyword
          runtime
          string
          dot
          identifier
          random
